내가 보기 위해 정리한 인텔리제이 단축키들📝  
(하나 하나 배울 때마다 내용을 추가하려고 한다.)  

| 분류 | 단축키 | 설명 | 
| :---: | :---: | :--- |
| 자동완성 | `Alt` + `Insert` | getter, setter 생성, 파일 추가 등 | 
| 명령어 실행 | `Shift` + `F10` | Run 실행 | 
|  | `Shift` + `F9` | 디버그 실행 |
